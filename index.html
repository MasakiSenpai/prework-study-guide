Prework Study Guide

HTML

  • The head element contains informaton about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element definds the title of a webpage which is important for SEO.
  • The section element defines a section in a document allowing content organized based on topics.
  • The meta element defines metadata such as the character set, decription, keywords, author, and viewport.
  • The br element creates a line of space or a break in the line.
  • The footer element appears at the bottom of the page, usually containing the author, conact, sitemap, and navigation.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • Padding indicates how much space we want around the content inside an element.
  • With a simple comma, you can combine multiple selectors into a single CSS rule, rather than repeating your self agian and agian

Git

  • pwd : displays the current working directory.
  • ls : displays the list of files and folders in the directory.
  • cd : navigattes between directories in path.
  • cd .. : navigates backwords, one level at a time in the path of dirctories.
  • mkdir : creates a new directory.
  • touch : creates a file inside a directory.
  • start : opens a file(on windows).
  • git status : checks which branch you're in.
  • git checkout -b branch-name : craetes a new branch and switched to it.
